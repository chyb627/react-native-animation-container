# react-native-animation-container

container with moving effect

## Installation

```sh
yarn add react-native-animation-container
```

## Usage

```js
import { FadeOutAnimation } from 'react-native-animation-container';

// ...

return(
    <FadeOutAnimation>
        {/* Text, View, ... etc */}
    </FadeOutAnimation>
)

```

## components that can be used

- FadeOutAnimation
- HorizontalAnimation
- VerticalAnimation
- AnimationButton
